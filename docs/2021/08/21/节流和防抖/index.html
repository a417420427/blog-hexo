节流和防抖

  • 2021-08-21
  • zhangfan

定义

  • 防抖(debounce): 触发高频事件后 n 秒内函数只会执行一次，如果 n 秒内高频事件再次被触发，则重新计算时间。
  • 节流(throttle): 高频事件触发，但在 n 秒内只会执行一次，所以节流会稀释函数的执行效率。

实现

  • 基础版
  • 保留原函数 this 指针
  • 异步函数
  • 延时前后是否触发(leading, trailing), 设置最大等待时间 ，这里情况比较复杂，暂时不单独讨论， 后面有时间再做详细的实现
